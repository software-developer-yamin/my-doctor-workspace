# My Doctor — Admin Panel

React SPA for platform administrators. Built with Vite; served statically by the Express backend.

## Stack

| Layer | Tech |
|-------|------|
| Build | Vite 8 |
| Language | TypeScript + React 19.2 |
| Routing | TanStack Router 1.168 (file-based) |
| Data | TanStack Query 5 + TanStack Table 8 |
| State | Zustand 5 (auth store) |
| UI | Tailwind CSS 4 + shadcn/ui (Radix) |
| Auth | Custom JWT stored in cookie (`thisisjustarandomstring`) |
| User management | Clerk (separate `/clerk/` routes only) |

## Commands

```bash
pnpm dev    # Vite dev server — backend must run separately (no proxy)
pnpm build  # Outputs to dist/ — must rebuild before backend picks up changes
```

**After build:** restart backend to serve updated `dist/`.

## Architecture

```
src/
  features/{domain}/
    components/   ← UI for this feature
    hooks/        ← useQuery / useMutation wrappers
    api/          ← Axios calls
    types/        ← TypeScript types
  routes/         ← File-based TanStack Router pages
    routeTree.gen.ts  ← AUTO-GENERATED — never edit manually
  stores/
    auth-store.ts ← Zustand auth (JWT from cookie)
  lib/
    api.ts        ← Axios instance + auth header injection
```

## Key Rules

- `routeTree.gen.ts` is auto-generated by TanStack Router — **never edit manually**
- Every route file: `export const Route = createFileRoute('...')({...})`
- Auth token in cookie `thisisjustarandomstring` via Zustand — use `useAuthStore()`
- Feature UI lives in `src/features/<domain>/` — route files stay thin
- Dead code tracked with Knip (`knip.config.ts`)
